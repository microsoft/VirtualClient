Redis

Redis is an open source (BSD licensed), in-memory, high-performance, distributed memory object caching system used as a database, cache, message broker and streaming engine.

  • Official Redis Documentation
  • Redis Github Repo

Two of the widely used tools for benchmarking performance of a Redis server include the Memtier and Redis benchmarks.

  • Memtier Benchmarking Tool
  • Memtier Benchmark Repo
  • Redis Benchmarking Tool

What is Being Measured?

Either the Memtier toolset or Redis benchmark is used to generate various traffic patterns against Redis instances. These toolsets performs GET and SET operations against the Redis server and provides throughput and latency percentile distributions.
